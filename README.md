# Kurodots

A custom CachyOS Hyprland rice inspired by clean glassy anime/minimal setups.

## Goal

- Dark glass theme
- Rounded Hyprland windows
- Floating Waybar
- Rofi app launcher
- Kitty terminal
- SwayNC notifications
- Wallpaper/theme scripts
- Safe install process

## Status

In development, reletivley functional.
